Overleaf 使用说明
================

上传步骤：
1. 打开 https://www.overleaf.com
2. 注册/登录
3. 点击 "New Project" → "Upload Project" → 上传此文件夹
4. 左上角 Menu → Compiler 选 "XeLaTeX"
5. 点 "Recompile" 编译

协作步骤：
1. 在 Overleaf 项目页面点右上角 "Share"
2. 输入队友的邮箱
3. 队友会收到邀请邮件，接受后即可实时协作
（免费版可免费邀请1个协作者，你们2人正好）

注意：不需要上传 .pdf、.aux、.log 等临时文件
