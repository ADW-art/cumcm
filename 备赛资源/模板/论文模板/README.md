# 分章节 LaTeX 论文模板

> 源自 Mrite Skill 的论文模板结构
> 每个章节独立一个 .tex 文件，通过 \include{} 在主文件中组装

## 文件结构

| 文件 | 说明 | 备注 |
|------|------|------|
| 论文.tex | 主文件 | 编译时运行 xelatex 论文.tex |
| 0.摘要.tex | 摘要 | 单独一页，不超过 900 字 |
| 1.引言.tex | 引言（问题背景+重述） | |
| 2.总体分析.tex | 总体分析 | 串联各问题的思路 |
| 3.模型假设.tex | 模型假设 | 每条一行 |
| 4.符号说明.tex | 符号说明 | 表格形式 |
| 5.模型的建立与求解.tex | 模型建立（主入口） | 用 \include{} 引入各问题 |
| 5.1.问题1的建立求解.tex | 问题1（入口） | 再 \include{5.1.1} 和 \include{5.1.2} |
| 5.1.1.分析与准备.tex | 问题1 - 分析+流程图 | |
| 5.1.2.建模与求解.tex | 问题1 - 建模+求解+结果 | |
| 5.2~5.4 | 问题2~4，同上结构 | 按实际题目数量增减 |
| 6.模型检验.tex | 灵敏度分析+误差分析 | |
| 7.模型评价.tex | 模型优缺点评价 | |
| 8.模型改进推广.tex | 改进方向 | |
| 9.参考文献.tex | 参考文献 | |
| 10.附录.tex | 附录（核心代码） | |
| format.cls | 格式类文件（CUMCM标准） | 定义了论文整体样式 |
| fonts/ | 思源宋体（用于编译） | 如果编译报字体错，改用系统字体 |

## 使用方法

```powershell
cd "E:\code\数学建模大赛\备赛资源\模板\论文模板"
xelatex 论文.tex
xelatex 论文.tex   # 跑两次确保交叉引用正确
```

## 优点

- 各章节独立文件，多人协作时可以各写各的章
- 编译时只改主文件的 \include{} 就能增减章节
- 问题多的题目（如4问）直接加 5.4、5.5 即可
- 模板自带 CUMCM 标准格式（format.cls），不需要自己调边距、字体
